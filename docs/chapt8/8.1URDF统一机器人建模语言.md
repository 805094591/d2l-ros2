# 8.1 URDF统一机器人建模语言

URDF（Unified Robot Description Format）统一机器人描述格式，URDF使用XML格式描述机器人文件。

一个最简单的URDF模型可以像下面这样

```xml
<?xml version="1.0"?>
<robot name="myfirst">
  <link name="base_link">
    <visual>
      <geometry>
        <cylinder length="0.6" radius="0.2"/>
      </geometry>
    </visual>
  </link>
</robot>
```
